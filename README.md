## Direction artistique / graphisme

- Style 8-bit / police pixelisée (type minecraft)
- Cartouches de jeux
- Les jeux tournent dans une borne d'arcade

## Documentation

- Installer les dépendances python
```shell
pip install -r requirements.txt
```
- Lancer le serveur
```shell
python3 main.py
```
